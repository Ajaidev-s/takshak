General Rules

1. Eligibility --> The competition is open to all participants
2. Generation Tool --> AI image-generation platform will be provided by the event coordinators
3. Prompts Limit --> The maximum number of prompts per level will be announced before each level starts
4. Time Limit --> Each level has a set time limit

Judging Criteria

--> Accuracy: How closely the generated image matches the provided reference.
--> Creativity: The innovation in using the prompt(s).
--> Efficiency: The number of prompts used to achieve the desired image (fewer prompts earn more points).
--> Prompt Length: The length of the prompt will be considered for evaluation
--> Adherence to Theme (Level 3): For Level 3, the creativity and relevance to the given theme.
All decisions made by the Judgement Committee are final, and no further questions or appeals will be accepted.

Competition Levels

1. Level 1 - Simple Image:
✷ Participants are provided with a simple image
✷ They must generate the closest matching image using no more than 5 prompts.
✷ Judging is focused on accuracy and prompt efficiency.
✷ Time : 20 Min
2. Level 2 - Hard Image:
✷ A more complex image is provided.
✷ Participants must generate a similar image using no more than 7 prompts.
✷ Judging focuses on how well participants break down the complexity of the image and recreate it with prompts.
✷ Time : 20 Min
3. Level 3 - Thematic Generation:
✷ A theme is provided instead of a reference image.
✷ Participants must create an image from scratch based on the theme using up to 10 prompts.
✷ Judging focuses on creativity, adherence to the theme, and how evocative the image is.
✷ Time : 20 Minutes

Disqualification

✧ If any malpractice detected the participants will be disqualified
✧ Submissions after the time limit will not be accepted.
✧ Exceeding prompt limit also disqualified

Good luck, and enjoy this intellectual journey! We hope it’s both challenging and fun!
